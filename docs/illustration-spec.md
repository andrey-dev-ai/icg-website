# Специфікація ілюстрацій для секції «Напрямки бізнесу» — ICG

## Призначення
Ця інструкція — для AI-генерації зображень (ChatGPT / Midjourney / інше). Кожне зображення буде використано як фон карточки напрямку бізнесу на сайті icg.in.ua.

---

## Розміри та пропорції

- **Співвідношення сторін:** 3:2 (горизонтальне)
- **Рекомендований розмір:** 900×600 px (мінімум 600×400 px)
- **Формат:** PNG з прозорим або темним фоном (#1a2035 або близький до #0a0f1e)

---

## Зони зображення (КРИТИЧНО)

Кожне зображення поділене на **3 горизонтальні зони:**

```
┌──────────────────────────────────┐
│                                  │
│      ЗОНА 1: ВЕРХНЄ ПОВІТРЯ      │  ← 15% висоти
│      (порожній простір, тільки    │     Нічого крупного!
│       зірки/іскри дрібні)        │     Дрібні декоративні елементи ок.
│                                  │
├──────────────────────────────────┤
│                                  │
│      ЗОНА 2: ІЛЮСТРАЦІЯ          │  ← 55% висоти
│      (основний контент —         │     Тут вся деталізація:
│       будівлі, техніка,          │     будинки, крани, труби і т.д.
│       конструкції)               │
│                                  │
├──────────────────────────────────┤
│                                  │
│      ЗОНА 3: НИЖНЯ ПОРОЖНЯ       │  ← 30% висоти
│      (тут буде текст поверх!)    │     ТІЛЬКИ горизонтальна лінія
│                                  │     землі/основи у верхній частині
│      ⚠️ НЕ МАЛЮВАТИ ТУТ         │     цієї зони, решта — порожній
│         НІЧОГО КРУПНОГО!         │     темний фон.
│                                  │
│      [ БУДІВНИЦТВО ]  ← текст    │     Текст буде білий/золотий,
│      [ опис дрібний ] ← текст    │     тому фон має бути темним
│                                  │
└──────────────────────────────────┘
```

### Деталі по зонах:

**Зона 1 (верх, 15%):**
- Порожній темний простір
- Дозволено: дрібні іскри, зірки, свічення (opacity < 0.5)
- Заборонено: великі елементи, текст, антени що виходять за межі

**Зона 2 (центр, 55%):**
- Основна ілюстрація — будівлі, техніка, конструкції
- Центрування горизонтальне — об'єкти розподілені рівномірно по ширині
- Стиль: gold line art (#b89a5a) на темному фоні
- Деталізація: вікна, двері, елементи конструкцій

**Зона 3 (низ, 30%):**
- ⚠️ ПОВИННА БУТИ ПОРОЖНЬОЮ (темний фон)
- Єдиний дозволений елемент: тонка горизонтальна лінія «землі» на ВЕРХНІЙ межі зони 3 (пунктир або суцільна, stroke 0.5-1px)
- Нижче лінії землі — НІЧОГО, тільки темний фон
- Тут сайт накладе текст (назву категорії + опис)

---

## Лінія землі / основи

**Обов'язковий елемент** на кожній ілюстрації:
- Горизонтальна лінія на рівні ~70% від верху зображення
- Стиль: пунктирна (`stroke-dasharray: 3 2`) або суцільна тонка
- Колір: #b89a5a, opacity 0.4-0.6
- Ширина: від краю до краю зображення
- Всі будівлі/об'єкти "стоять" на цій лінії

**Всі 5 зображень повинні мати лінію землі НА ОДНОМУ РІВНІ** (70% від верху). Це критично для візуальної цілісності ряду карточок.

---

## Стиль ілюстрацій

- **Техніка:** Line art / wireframe на темному фоні
- **Колір ліній:** #b89a5a (золотий ICG) — ОДИН колір для всіх
- **Фон:** #1a2035 або максимально близький до #0a0f1e (deep navy)
- **Stroke width:** 0.5-1.5px (тонкі лінії = преміальний вигляд)
- **Заливка:** мінімальна, тільки акцентні елементи (opacity 0.03-0.15)
- **Ефекти:** дозволені дрібні іскри/свічення на ключових точках (opacity < 0.5)
- **Текстура:** легка зернистість фону ок, але НЕ надто виражена

```
border: 1px solid rgba(160, 170, 195, 0.1)
border-radius: 14px
```

## 5 ілюстрацій — що малювати

### 1. Будівництво (construction.png)
**Об'єкти в Зоні 2:**
- Зліва: багатоповерхівка в лісах (будівельні ліси видно)
- По центру-праворуч: менший будинок/котедж
- Справа: будівельний кран (мачта + стріла + підвіс)
- Дрібні деталі: вікна, балкони, каска на парканчику

### 2. Нерухомість (realestate.png)
**Об'єкти в Зоні 2:**
- Три будівлі різної висоти (житлова, офісна висотна, середня)
- Ключ або символ угоди поруч з будівлями
- Дерева/кущі між будівлями (дрібні)

### 3. Виробництво (manufacturing.png)
**Об'єкти в Зоні 2:**
- Заводський цех з двоскатним дахом
- 2-3 димові труби з кільцями диму
- Конвеєрна лінія з коробками продукції
- Блок контролю якості (✓)

### 4. Корисні копалини (minerals.png)
**Об'єкти в Зоні 2:**
- Бурова вишка по центру (трикутна конструкція)
- Хвилясті лінії геологічних порід під вишкою
- Вагонетка з породою збоку
- Документ/ліцензія збоку

### 5. Реалізовані проєкти (projects.png)
**Об'єкти в Зоні 2:**
- 3-4 різні побудовані об'єкти в ряд
- Над кожним — кружок з галочкою «завершено»
- Внизу — лінія прогресу з точками
- Документ-портфоліо справа
